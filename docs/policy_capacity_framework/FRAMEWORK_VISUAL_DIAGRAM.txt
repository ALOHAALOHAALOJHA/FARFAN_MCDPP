│                          MATHEMATICAL MODEL SUMMARY                          │
└──────────────────────────────────────────────────────────────────────────────┘

1. BASE CAPACITY SCORE (Individual Method)
   ═════════════════════════════════════════
   C_base = 0.4×E_weight + 0.35×L_weight + 0.25×O_weight
   
   Range: 0.0 - 2.0

2. AGGREGATION WEIGHT (Cross-Level)
   ═════════════════════════════════
   W_agg = 0.85 × exp(-0.15 × Δ_level)
   
   Where Δ_level = distance between capacity levels
   
3. EQUIPMENT CONGREGATION MULTIPLIER (Synergy)
   ════════════════════════════════════════════
   M_equip = 1 + 0.3 × ln(1 + n_skills) × (ρ - 1)
   
   Increases with number of combined skills

4. SYSTEMIC CAPACITY SCORE (Per Type)
   ════════════════════════════════════
   C_systemic = [Σ(C_base × W_agg × M_equip)] / N
   
   Normalized by number of methods

5. INTEGRATED CAPACITY INDEX (Overall)
   ══════════════════════════════════
   ICI = Σ(w_skill × Σ(w_level × C_type)) / 9
   
   Level weights: Individual=0.25, Org=0.35, Systemic=0.40
   Result: ICI = 0.7885
